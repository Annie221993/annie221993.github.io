## ✅ Temas vistos en clase

En las primeras sesiones del curso aprendimos conceptos fundamentales para iniciar en el mundo de la programación. Lo primero fue entender *qué es un programa*, que básicamente es un conjunto de instrucciones que le damos a un computador para que realice una tarea específica. También vimos que esas instrucciones deben ser claras, ordenadas y lógicas para que la máquina pueda ejecutarlas sin errores.

Otro tema importante fueron las *variables*, que son espacios en memoria donde podemos guardar información. Una variable tiene un nombre y un valor, y puede contener números, texto o datos más complejos dependiendo del lenguaje de programación. En Python, por ejemplo, no necesitamos indicar qué tipo de dato tendrá la variable; simplemente se asigna y Python lo interpreta.

También hablamos sobre los *tipos de datos*, como enteros, decimales, cadenas de texto y booleanos, que son esenciales para poder trabajar con información de manera correcta. Además, comprendimos por qué es tan importante escribir código limpio, ordenado y legible para nosotros mismos y para otros programadores.
