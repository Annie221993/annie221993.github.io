# Tarea 2 — Unidad 1 ## Actividad Práctica: Análisis de una Cadena en Python Este programa recibe una cadena ingresada por el usuario y cuenta cuántas letras mayúsculas, minúsculas y dígitos contiene. ### Código en Python ```python cadena = input("Ingrese una cadena de caracteres: ") mayusculas = 0 minusculas = 0 digitos = 0 for caracter in cadena: if caracter.isupper(): mayusculas += 1 elif caracter.islower(): minusculas += 1 elif caracter.isdigit(): digitos += 1 print("Mayúsculas:", mayusculas) print("Minúsculas:", minusculas) print("Dígitos:", digitos) ``` ### Explicación - Se pide al usuario una cadena de texto. - Se recorre cada carácter y se verifica: - Si es mayúscula. - Si es minúscula. - Si es un número. - Finalmente se imprimen los totales encontrados.
